# Grow Tent Monitor

Web app for monitoring a grow tent.

## Tech:
- React.js frontend
  - Tailwind CSS
  - Recharts
  - Formik
  - Phosphor Icons
  - FullCalendar
- Node.js backend
  - Fastify
  - MariaDB (MySQL)
- ESP32 chips
  - ESP32 Wifi Chip
    - DHT11 Sensor
  - ESP32 Cam
    - MJPEG stream
